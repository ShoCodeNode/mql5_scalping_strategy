# 戦略詳細ドキュメント

## 🎯 戦略概要

**アダプティブ・スキャルピング戦略**は、ATRベースの動的トレーリングストップを搭載したマルチタイムフレーム分析によるスキャルピング手法です。

## 📊 分析フレームワーク

### マルチタイムフレーム構成

```
上位時間軸 (M5) - トレンド方向確認
├── EMA20によるトレンド判定
├── 価格位置の確認
└── 全体的な市場方向性

下位時間軸 (M1) - エントリータイミング  
├── EMA(5,10,20)のアライメント
├── RSI(14)の過買い・過売り
├── MACD(12,26,9)のクロス
└── 価格アクションパターン
```

## 🔍 エントリー条件

### 1. トレンド確認 (M5)

**買いトレンド条件**:
- 現在価格 > EMA20(M5)
- EMA20(M5)[0] > EMA20(M5)[1] (上昇中)

**売りトレンド条件**:
- 現在価格 < EMA20(M5)
- EMA20(M5)[0] < EMA20(M5)[1] (下降中)

### 2. EMAアライメント (M1)

**買いシグナル**:
```
EMA5 > EMA10 > EMA20
短期が長期を上抜けている状態
```

**売りシグナル**:
```
EMA5 < EMA10 < EMA20  
短期が長期を下抜けている状態
```

### 3. RSI条件

**買い条件**:
- RSI ≤ 30 (売られ過ぎ)
- RSI[0] > RSI[1] (反転開始)

**売り条件**:
- RSI ≥ 70 (買われ過ぎ)
- RSI[0] < RSI[1] (反転開始)

### 4. MACDクロス

**買いシグナル**:
- MACD[0] > Signal[0]
- MACD[1] ≤ Signal[1]
- (ゴールデンクロス)

**売りシグナル**:
- MACD[0] < Signal[0]  
- MACD[1] ≥ Signal[1]
- (デッドクロス)

### 5. 価格アクション確認

**買いパターン**:
- **ハンマー/ピンバー**: 下影 > 実体×2, 上影 < 実体
- **陽線エンガルフィング**: 前陰線を包む大陽線

**売りパターン**:
- **シューティングスター**: 上影 > 実体×2, 下影 < 実体  
- **陰線エンガルフィング**: 前陽線を包む大陰線

## 🎯 シグナル強度計算

```
シグナル強度 = 各条件の合計点数

トレンドアライメント: 30点
EMAアライメント: 25点  
RSI条件: 20点
MACD条件: 15点
価格アクション: 10点

合計: 100点満点
取引閾値: 80点以上
```

## 🛡️ リスク管理

### ポジションサイジング

```
ロット計算式:
Lot = (口座残高 × リスク%) / (SL pips × pip値)

例: 残高$10,000, リスク1%, SL 10pips
Lot = ($10,000 × 1%) / (10 × $1) = 10ロット
```

### ストップロス

**初期SL**: 10pips固定
**根拠**:
- スキャルピングに適したタイトなSL
- スプレッドコストとのバランス
- ノイズによる誤ヒット回避

## 🔄 ATRベース・トレーリング

### トレーリング開始条件

```
開始条件: 利益 ≥ ATR × 1.5

例: ATR = 8pips の場合
開始利益 = 8 × 1.5 = 12pips
```

### トレーリング間隔

```
追従間隔 = ATR × 0.8

例: ATR = 8pips の場合  
間隔 = 8 × 0.8 = 6.4pips
```

### 最小利益保証

```
最小保証 = ATR × 0.5

例: ATR = 8pips の場合
最小利益 = 8 × 0.5 = 4pips
```

### 動作例

```
シナリオ: EUR/USD 買いポジション
エントリー: 1.1000
ATR: 0.0008 (8pips)

1. +12pips利益時 → トレーリング開始
   SL: 1.1000 + 0.0004 = 1.1004 (+4pips保証)

2. +18pips利益時 → SL更新  
   SL: 1.1000 + 0.0010 = 1.1010 (+10pips保証)

3. +24pips利益時 → SL更新
   SL: 1.1000 + 0.0016 = 1.1016 (+16pips保証)
```

## 🕐 時間フィルター

### 取引セッション

**ロンドンセッション**:
- 時間: 16:00-20:00 JST
- 特徴: 高ボラティリティ
- 対象ペア: EUR/USD, GBP/USD

**ニューヨークセッション**:  
- 時間: 22:00-02:00 JST
- 特徴: 高流動性
- 対象ペア: USD/JPY, EUR/USD

**重複時間**:
- 時間: 22:00-01:00 JST  
- 特徴: 最高の流動性とボラティリティ
- 最適取引時間

### 回避時間

**金曜日遅い時間**: 22:00以降新規取引停止
**理由**: 週末ギャップリスク回避

## 📈 パフォーマンス最適化

### 通貨ペア選択

**優先順位**:
1. **EUR/USD**: 最高の流動性、安定したスプレッド
2. **GBP/USD**: 高ボラティリティ、大きな値幅
3. **USD/JPY**: アジア時間でも取引可能

### 市場環境適応

**トレンド相場**:
- EMA期間を長めに調整
- トレーリング開始を早めに

**レンジ相場**:
- RSI閾値を厳しく設定  
- 利確目標を短めに

### パラメーター調整指針

```
勝率が低い場合:
- RSI閾値を厳しく (25/75)
- シグナル強度閾値を上げる (85点以上)

利益が少ない場合:  
- トレーリング間隔を広げる (ATR×1.0)
- 初期TP を併用 (15pips)

ドローダウンが大きい場合:
- リスク%を下げる (0.5%)
- 最大ポジション数制限
```

## 🎛️ 高度な設定

### 市場環境フィルター

**ボラティリティフィルター**:
- ATR < 平均×0.5 → 取引停止
- ATR > 平均×2.0 → リスク減少

**スプレッドフィルター**:  
- スプレッド > 3pips → 取引停止
- 重要指標発表時の拡大対応
